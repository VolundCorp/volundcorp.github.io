Politique de Confidentialité de Volund

Dernière mise à jour : 07/01/2025

Cette Politique de Confidentialité décrit les informations que le bot Discord "Volund" collecte, comment elles sont utilisées, et vos droits concernant ces informations.

1. Données collectées

Lorsque vous utilisez Volund, les données suivantes peuvent être collectées :
  • Identifiant Discord des utilisateurs (User ID) : Ces informations permettent au bot d’interagir avec les utilisateurs de manière personnalisée.
  • Identifiant des serveurs (Server ID) : Ces informations permettent d’identifier sur quel serveur le bot est utilisé.
  • Identifiant des canaux de discussion (Channel ID) : Ces informations permettent de traiter les commandes ou interactions spécifiques au bon endroit.

2. Utilisation des données

Les données collectées sont utilisées exclusivement pour le fonctionnement du bot et pour fournir les fonctionnalités suivantes :
  • Répondre aux commandes effectuées par les utilisateurs.
  • Identifier le contexte d'utilisation (serveur, canal).
Volund ne stocke ni n’analyse vos messages ou contenu partagé dans les canaux Discord.

3. Partage des données

Les données collectées par Volund ne sont pas partagées avec des tiers, sauf si cela est requis par la loi.

4. Durée de conservation des données

Les données sont conservées uniquement pendant la durée nécessaire pour fournir les services du bot. Si le bot est retiré d’un serveur, toutes les données associées à ce serveur seront supprimées.

5. Vos droits

Conformément au RGPD (Règlement Général sur la Protection des Données) :
  • Vous avez le droit d’accéder, de modifier ou de demander la suppression des données personnelles collectées.
  • Vous pouvez également demander des informations sur la manière dont vos données sont utilisées.
Pour toute demande ou question concernant vos données, veuillez nous contacter à : contact@volund.gg

6. Hébergement des données

Les données collectées et utilisées par Volund sont hébergées sur une infrastructure fournie par OVHcloud, une entreprise basée en Europe et conforme au RGPD. OVHcloud garantit un haut niveau de sécurité et de conformité pour les données hébergées sur ses serveurs. Pour en savoir plus, vous pouvez consulter leur politique de confidentialité ici : https://www.ovhcloud.com/fr/personal-data-protection/.

7. Sécurité des données

Volund prend des mesures raisonnables pour protéger les données collectées contre tout accès non autorisé, perte ou divulgation. Cependant, aucune méthode de transmission ou de stockage n’est totalement sécurisée.

8. Modifications de cette politique

Cette politique peut être mise à jour à tout moment. En cas de modification importante, une notification sera communiquée via les serveurs utilisant Volund.

9. Responsable du traitement

Le responsable du traitement des données est : Lilierd lilierd@volund.gg En utilisant Volund, vous acceptez cette politique de confidentialité.
