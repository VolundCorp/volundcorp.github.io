Slash commands

Volund has a lot of slash commands to help you. You can see all of them by clicking on the app button. From setup to information commands, you can use all these commands:

Action commands

  • /setup - This is the first command to execute when you have just installed Volund. Optionnally, you can add a custom name to it. For more information about the forge, please see this guide.
    Admin only
  • /setup-mine - This is the second command to execute when you have just installed Volund. It will create a specific channel for the mine of your server. For more information about the mine, please see this guide.
    Admin only
  • /unlock - Volund has a lot more features than what you think! In fact, you can unlock them with this command which will display a list of all the features you haven't unlocked yet. Of course, each feature has a cost of ingots that you could find the mine.
    Admin only
  • /remove-category - Of course you can delete manually channels created by Volund, but it won't be removed from the categories count. You can just execute the command to be displayed a list of all the categories where you can choose the one to delete or you can add the ID of the role that has been created with it.
    Admin only
  • /edit-category - If you want to edit a category, it is possible to do so by using this command. First, it will ask you to choose which category you want to edit. Then, it will display the category modifying modal, allowing you to customize it.
    Admin only
  • /edit-lobby - If you want to edit a lobby, it is possible to do so by using this command. First, it will ask you to choose which lobby you want to edit. Then, it will display the lobby modifying modal, allowing you to customize it (change its template/name).
    Admin only
  • /invest - It allows every user of your server to put some ingots in your server's balance.
  • /actually - You can use this command to make your message look smarter ‚òùÔ∏èü§ì.

Information commands

  • /info - To get information about your server. It will display a message with the amount of ingots this server has and the list of unlocked features.
    Admin only
  • /balance - To get your balance of ingots you earned by mining.
  • /about - To get general information about Volund.

Temporary voice channels | Voice lobbies

Once the forge has been set, it is possible for you to create voice lobbies. These channels are meant to create as many voice channels as you want by simply joining them.
