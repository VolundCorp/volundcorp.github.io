Slash commands

Volund has a lot of slash commands to help you. You can see all of them by clicking on the app button. From setup to information commands, you can use all these commands:

Action commands

  • /setup - This is the first command to execute when you have just installed Volund. Optionnally, you can add a custom name to it. For more information about the forge, please see this guide.
    Admin only
  • /setup-mine - This is the second command to execute when you have just installed Volund. It will create a specific channel for the mine of your server. For more information about the mine, please see this guide.
    Admin only
  • /unlock - Volund has a lot more features than what you think! In fact, you can unlock them with this command which will display a list of all the features you haven't unlocked yet. Of course, each feature has a cost of ingots that you could find the mine.
    Admin only

Information commands

  • /info - To get information about your server. It will display a message with the amount of ingots this server has and the list of unlocked features.
    Admin only
  • /balance - To get your balance of ingots you earned by mining.
  • /about - To get general information about Volund.
