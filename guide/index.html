Forge

By using /setup, you'll create Volund's forge.

The forge is the foundation of Volund. Thanks to the it, you will be able to perform several actions linked to the bot:

  • Create a category: These custom sections can contain as many text channels as you want, a news channel and a voice lobby. It is up to you to choose among these different channels to put in your category. When you click on the ‚öíÔ∏è button, you'll be asked to fill in a form with :
    • Category's name
    • Number of text channels
    • Whether you want a news channel or not
    • Whether you want a voice lobby or not
  • Create a voice lobby: This type of voice channel is a bit special. Once you connect to it, it will create a temporary voice channel and move you to it.
  • Upgrade your own tool: In order to mine with more efficiency in Volund's mine, you can upgrade your own tool. For more information, see here.
  • Increase categories limit: You can't create infinite categories but you can increase this limit just by paying some ingots gathered in the mine.
  • Set welcome message: Volund is capable of welcoming your newcomers with custom message.
    You'll have to unlock it with /unlock first.
  • Increase mine chance: each time you upgrade your mine, it will gain 5% more chance to find ores!

Mine

By using /setup-mine, you'll create Volund's mine. This mine is actually a mini-game where you can dig and try to harvest ores with the ‚õèÔ∏è button. To do so, you have a personal tool that you can upgrade with the forge. Each time you harvest an ore, you'll earn 1 ingot per level of your tool.

Before finding any ore, you'll have to dig in the mine. Each time you try to dig, you have a chance to find 5 ores. This chance is 50% when Volund has just been installed but can be increased by 5% with the forge. Once you tried to dig, if you don't find ore, then you'll have to wait 60 seconds before being able to dig again. If you succeed and find ore, then you'll be able to harvest it every 30 seconds until the vein runs out.

The ingots (or gold) you earn by mining are stored in your personal wallet. You can use it to upgrade your own tool in the forge or invest it in the server using the /invest command.

Unlocking features

Thanks to the /unlock command, you'll be able to unlock locked features.

For example, if you want to set a customized welcome message, you will first have to unlock this feature. Unlocking a feature has a cost and this cost will be applied on the server's wallet. If you want to pay for it, you can just check the amount needed and then use the /invest command.

Investing in a server

Some features of Volund have a cost, whether for unlocking or for use. In order to allow the server to grow thanks to Volund, you will have to invest your ingots you got by mining in the server's wallet. Simply use /invest command and a form will ask you how much you want to invest.
